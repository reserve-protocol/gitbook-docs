# Fees

### Fees & Revenue <a href="#fees--revenue" id="fees--revenue"></a>

Index DTFs generate revenue through two primary fee mechanisms:

1. **TVL Fee**: A fee based on the Total Value Locked in the protocol
2. **Mint Fee**: A fee applied when users mint new DTF tokens

Both fees are collected and distributed in the form of the DTF token itself to one or multiple recipients, as determined by governance. A platform fee is applied to both the TVL and Mint fees before they are distributed to recipients.
